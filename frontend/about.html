About the Project

This project focuses on detecting Polycystic Ovarian Syndrome (PCOS) from ultrasound images using a custom-built Convolutional Neural Network (CNN) along with Explainable AI techniques.

  • Model Accuracy: 99.87% using a custom CNN
  • Explainability: LIME highlights important regions for prediction
  • Comparison: Outperforms pretrained models like ResNet and DenseNet

Technologies Used

TensorFlow, Keras, OpenCV, LIME, Matplotlib

Why It Stands Out

The model not only gives a prediction but explains its decision visually. This transparency builds trust in AI and makes the system useful for real clinical settings.
